* data systems are evolutioning

  We typically think of databases, queues,caches,etc. as being very different categories of tools. However, many new tools for data storage and processing have emerged in recent years. There are datastores that are also used as message queues(Redis), and there are message queues with database-like durability guarantees(kafka). The boundaries between the categories are becoming blurred.
